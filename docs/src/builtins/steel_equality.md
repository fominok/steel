# steel/equality
### **=**
Compares two numbers for equality.

(= left right) -> bool?

* left : number? - The first number to compare.
* right : number? - The second number to compare.

#### Examples
```scheme
> (= 1 1) ;; => #t
> (= 1 1.0) ;; => #t
> (= 2 3) ;; => #f
